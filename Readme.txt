감정데이터와 생체데이터 간의 연관성 분석
- 다양한 상황에서 추출한 감정데이터와 그에 따른 생체신호[ECG,EEG,EDA] 데이터간의 연관성 분석을 통해 더 신뢰적인 감정 데이터 지표를
  제공

Association : 연관성 분석 관련, Weka Tool 이용
ECG : 심박수 측정 관련코드, fitbit 팔찌 이용
EDA : 피부전도 측정 관련코드, bitalino 멀티센서 사용
EEG : 뇌파 측정 관련코드, bitalino 멀티센서 사용
EMG : 근전도 측정 관련코드, bitalino 멀티센서 사용


bitalino : http://bitalino.com/en/
